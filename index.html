Engineering Challenge

The existing actuator system lacked absolute positioning capability, relying on basic on/off control. This limited precision for advanced milling operations requiring repeatable, numeric positioning with sub-millimetre accuracy. The challenge was to retrofit full numeric control whilst maintaining system reliability and creating an intuitive operator interface.

Solution & Implementation

System Integration

Interfaced SMC JXC918 controller with Unitronics PLC via Modbus RTU communication protocol, enabling real-time position feedback and motion control commands.

Calibration & Homing

Developed robust calibration routines including reference positioning, zero-point establishment, and error recovery logic to ensure consistent absolute positioning.

Motion Control Logic

Programmed ladder logic sequences for jog control (manual positioning), automatic mode (pre-set positions), speed ramping, and acceleration profiles for smooth motion.

HMI Development

Designed intuitive operator interface featuring position display, jog controls, auto-mode selection, diagnostic readouts, and emergency stop/reset functions.

Error Handling

Implemented comprehensive fault detection including over-travel protection, communication timeout monitoring, and position deviation alarms with clear operator guidance.

Engineering Challenge

Manual CNC machine tending was creating production bottlenecks, requiring operators to load/unload parts every 3-4 minutes. This limited throughput, introduced quality variation from inconsistent part placement, and prevented operators from managing multiple machines. The challenge was to automate the tending process whilst ensuring part quality, safety compliance, and minimal disruption to existing production.

Robot Platform
Universal Robots UR10 collaborative robot
End Effector
Custom dual-sided vacuum gripper
Vision System
Machine vision for part detection & QC
Cycle Time

Solution & Implementation

Gripper Design & Fabrication

Designed dual-sided vacuum gripper using Fusion 360, fabricated mounting brackets on CNC mill, and integrated vacuum generators with pressure sensing for reliable part handling.

Robot Programming

Developed URScript programs for pick-place sequences, coordinated multi-point movements, and implemented force/torque monitoring for safe CNC door operation and part insertion.

Machine Vision Integration

Configured vision system for part presence detection, orientation verification, and basic quality inspection (surface defects, dimensional checks) with pass/fail sorting.

CNC-Robot Communication

Interfaced UR10 with CNC machine PLC via digital I/O for cycle coordination, door interlocking, and status monitoring ensuring safe, synchronized operation.

Safety Implementation

Configured collaborative safety zones, reduced robot speed in operator areas, and installed emergency stops with risk assessment documentation meeting UK HSE requirements.
